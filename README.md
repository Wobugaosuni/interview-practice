## 使用

方法一. npm start 启动
```bash
## 安装项目依赖
npm install

## 开启mock服务，使用的是3000的端口
npm run mock

## 启动项目，使用的是8000的端口
npm start
```

**建议使用 storybook 启动
方法二. storybook启动
```bash
npm run storybook
```
